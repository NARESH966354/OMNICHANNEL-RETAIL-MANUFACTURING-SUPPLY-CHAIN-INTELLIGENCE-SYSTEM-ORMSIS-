## Overview
Built an end-to-end Excel dashboard to analyze supply chain performance including inventory, warehouse operations, and demand forecasting.

## Key Features
- Inventory & stock analysis
- Warehouse performance tracking
- Supplier insights
- Demand forecasting using moving average

## Dashboard
- KPI Cards (Revenue, Profit, Inventory Turnover)
- Charts for trend & forecast analysis
- Slicers for interactivity

## Key Insights
- Forecast accuracy is low (~39%)
- High demand variability observed
- Forecasting model needs improvement

## Tools Used
- Microsoft Excel
- Pivot Tables
- Charts & Slicers
